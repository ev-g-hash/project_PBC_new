имеется приложение main_shop в котором:
    имеются файлы:
    	index.html, aboutas.html, forms.html, readmore.html

    forms.py содержит следующие поля:
	name, email, phone, message

    urls.py содержит следующие пути:
	aboutas/, readmore/, forms/  

    views.py содержит обязательное условие в свох представлениях:   
        def index - render shop/index.html
	def aboutas - render shop/aboutas.html
	def readmore - render shop/readmore.html
	def forms - request.method == 'POST' всех полей forms.py
	после заполнения и отправки формы пользователем выводится сообщение - Ожидайте в ближайшее время мы с вами свяжемся!
